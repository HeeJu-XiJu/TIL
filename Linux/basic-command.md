# basic command
>리눅스 기본 명령어 정리

- `pwd` (printworking directory) : 현재 작업중인 폴더 경로를 출력

- `ls` (list) : 현재 작업중인 폴더 내에 있는 파일/폴더를 출력

- `ls-a` (optional) : 숨김처리되어있는 파일/폴더까지 출력
     - `.` : 현재 나의 폴더
     - `..` : 현재 나의 폴더 기준 상위 폴더

- `cd` (change directory) : 폴더 이동
    - `cd ..` : 상위 폴더로 이동
    - `cd ~` : 최상위 폴더로 이동
    - `cd <폴더명>` : <폴더명>으로 이동
    - `cd <폴더명 첫글자>` : 폴더명 첫글자 입력 후 탭탭을 누르면 첫글자가 포함된 폴더를 모두 출력해줌

- `mkdir` (make directory) : 폴더 생성
    -`mkdir <폴더이름>` : <폴더이름>으로 새로운 폴더 생성

- `touch` : 파일 생성
    - `touch <파일이름>` : <파일이름>으로 새로운 파일 생성

- `rm` (remove) : 파일 삭제
    - `rm <파일이름>` : <파일이름> 파일 삭제 
    - `rm -r <폴더이름>` : <폴더이름>에 해당하는 폴더 삭제 (recursive)
